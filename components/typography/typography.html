Heading Utilities

Heading are titles or subtitles that you want to display on a webpage.

Heading starts from h1 to h6. You can use it by giving the tag names h1, h2, h3, h4, h5, h6.

Font Size Utilities

Font Size Utilitiy is used to define the size of text. It start from 4px to 32px and has 8 sizes.

Use the class name according to your requirement. Refer the below code.

Font Size 1 has size of 4px

Font Size 2 has size of 8px

Font Size 3 has size of 12px

Font Size 4 has size of 16px

Font Size 5 has size of 20px

Font Size 6 has size of 24px

Font Size 7 has size of 28px

Font Size 8 has size of 32px


        <p class="font-size-1">Font Size 1 has size of 4px</p>
        <p class="font-size-2">Font Size 2 has size of 8px</p>
        <p class="font-size-3">Font Size 3 has size of 12px</p>
        <p class="font-size-4">Font Size 4 has size of 16px</p>
        <p class="font-size-5">Font Size 5 has size of 20px</p>
        <p class="font-size-6">Font Size 6 has size of 24px</p>
        <p class="font-size-7">Font Size 7 has size of 28px</p>
        <p class="font-size-8">Font Size 8 has size of 32px</p>

Grey Text

To color the text gray,use the class name text-gray.

This text is the Grey Text


        <p class="font-size-8 font-bold text-gray">This text is the Grey Text</p>

Text Alignment

To align the text in left which is also the default, use the class text-left.

To align the text in center, use the class text-center.

To align the text in right, use the class text-right.

This text is align to the left of container

This text is align to the center of container

This text is align to the right of container


        <p class="font-size-5 text-left">This text is align to the left of container</p>
        <p class="font-size-5 text-center">This text is align to the center of container</p>
        <p class="font-size-5 text-right">This text is align to the right of container</p>
