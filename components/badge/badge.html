Badge

Badge is used to highlight an Items Status.

Badge on Avatar

In Badge on you can use the class badge-wh-1 for the size of badge.Starts from 1 and end to 5.

For badge color you can use two colors primary-badge-color, secondary-badge-color, offline-badge-color and online-badge-color

For Round Avatar, Image or Icon it has 4 position bottom right, top right, bottom left and top left. Can use this class badge-position-round-br, badge-position-round-tr, badge-position-round-bl and badge-position-round-tl. 


        <!-- badge position bottom-right for round icon/avatar -->
        <figure class="badge-container">
            <img src="../../assests/avatar.jpg" alt="avatar" class="avatar-sm border-round">
            <span class="badge badge-wh-1 badge-position-round-br offline-badge-color"></span>
        </figure>

        <!-- badge position top-right for round icon/avatar -->
        <figure class="badge-container">
            <img src="../../assests/avatar.jpg" alt="avatar" class="avatar-md border-round">
            <span class="badge badge-wh-2 badge-position-round-tr secondary-badge-color"></span>
        </figure>

        <!-- badge position bottom-left for round icon/avatar -->
        <figure class="badge-container">
            <img src="../../assests/avatar.jpg" alt="avatar" class="avatar-lg border-round">
            <span class="badge badge-wh-2 badge-position-round-bl offline-badge-color"></span>

Badge on Icons

In Badge on Icon you can use the class badge-wh-1 for the size of badge.Starts from 1 and end to 5.

For badge color you can use two colors primary-badge-color, secondary-badge-color, offline-badge-color and online-badge-color

For Square Avatar, Image or Icon it has 4 position bottom right, top right, bottom left and top left. Can use this class badge-position-square-br, badge-position-square-tr, badge-position-square-bl and badge-position-square-tl.

If you input text in badge dont forget to use class
