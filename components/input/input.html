Input

Inputs are used to take information from the user, and then save the data to the server, accordingly.

Input

Default input has label and input in it. Only you have to copy the below code to use the input.

You can use any type of input like email, password, textarea, etc only you have to change the type in input tag.


                        <div class="input-container">
                            <label class="input-label">Username*</label>
                            <input type="text" placeholder="Enter your username" class="input">
                        </div>

Error Input

In Error Input you have to put two classes in default input.

label-error class in label tag

input-error class in input tag


                            <div class="input-container">
                            <label class="input-label label-error">Password*</label>
                            <input type="text" placeholder="Enter your password" class="input input-error">
                        </div>
