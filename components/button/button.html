Button

Buttons are nothing but call to action.We have Standard Buttons, Outline Buttons, Icon Buttons and Floating Button.

Standard Buttons

                        <button class="btn primary-btn">Primary Button</button>
                            <button class="btn secondary-btn">Secondary Button</button>

Outline Buttons

                        <button class="btn primary-outline-btn">Primary Button</button>
                            <button class="btn secondary-outline-btn">Secondary Button</button>

Link Buttons

Primary Button Secondary Button 
                        <a class="btn primary-btn">Primary Button</a>
                            <a class="btn secondary-btn">Secondary Button</a>

Icon Buttons

                        <button class="btn primary-outline-btn"><i class="fas fa-shopping-cart mr-3"></i>Add to Cart</button>
                            <button class="btn secondary-btn"><i class="fas fa-heart mr-3"></i>Add to Wishlist</button>

Floating Buttons

                        <button class="btn primary-outline-btn"><i class="fas fa-shopping-cart fa-2x"></i></button>
                            <button class="btn secondary-outline-btn"><i class="fas fa-heart fa-2x"></i></button>
