Button

Buttons are nothing but call to action.

Standard Buttons


        <!--Standard Primary Button-->
        <button class="btn primary-btn">Primary Button</button>

        <!--Standard Secondary Button-->
        <button class="btn secondary-btn">Secondary Button</button>

Outline Buttons


        <!--Outline Primary Button-->
        <button class="btn primary-outline-btn">Primary Button</button>

        <!--Outline Secondary Button-->
        <button class="btn secondary-outline-btn">Secondary Button</button>

Link Buttons

Link Primary Button Link Secondary Button 

        <!--Link Primary Button-->
        <a class="btn primary-btn">Link Primary Button</a>

        <!--Link Secondary Button-->
        <a class="btn secondary-btn">LinkSecondary Button</a>

Icon Buttons


        <!--Icon Buttons-->
        <button class="btn primary-outline-btn"><i class="fas fa-shopping-cart"></i>Add to Cart</button>
        <button class="btn secondary-btn"><i class="fas fa-heart"></i>Add to Wishlist</button>

Floating Buttons


        <!--Floating Buttons-->
        <button class="btn primary-outline-btn"><i class="fas fa-shopping-cart fa-2x"></i></button>
        <button class="btn secondary-outline-btn"><i class="fas fa-heart fa-2x"></i></button>
