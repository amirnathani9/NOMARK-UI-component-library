Responsive-Image

Responsive Image Components are used to dsiplay Images on websites, which are screen responsive.

Full-Width Image

Full-Width Image is set according to your container width. To use the Full-Width responsive image, add responsive-img class in your img tag.


                            <img src="../../assests/image.jpg" alt="watch" class="responsive-img">

Rounded Image

Rounded Image work according to your container width. To use rounded image add round-img class in your img tag.


                            <img src="../../assests/image.jpg" alt="watch" class="round-img">
