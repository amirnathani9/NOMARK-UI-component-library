Responsive-Image

Responsive Image Components are used to display images on websites, which are screen responsive.

Full-Width Image

Full-Width Image is set according to your container width. To use the full-width responsive image, add responsive-img class in your img tag.


        <img src="put the image path here" alt="add alt text" class="responsive-img">

Rounded Image

Rounded Image work according to your container width. To use rounded image add border-round class in your img tag.


        <img src="put the image path here" alt="add alt text" class="border-round">
