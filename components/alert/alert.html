Alert

Alerts are used to attract user's attention for important information without interrupting the user's flow.

Standard Alerts

Standard Alert has 5 types Primary, Secondary, Success, Warning and Danger.

You can use any class primary-alert, secondary-alert, success-alert, warning-alert and danger-alert.

You can also use padding and marging utility classe to add margin and padding.

Alert with Icons

Icon Alert has 5 types Primary, Secondary, Success, Warning and Danger.

You can use any class of Standard Alerts and Outline Alerts.

Only you have to add the i tag of font-awesome in your alert tag.
