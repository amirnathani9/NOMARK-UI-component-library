Responsive Navbar

Responsive navbar is compatable with all devices. If screen sizes decreases the nav menu will store in the humburger icon.

To see the responsiveness decrease the size of the screen.

To use this navbar copy the below html code and javascript also

  • Account
  • Wishlist
  • Cart

HTML


        <nav class="navbar-container fixed z-5">
        <div class="navbar-logo-container">
            <a href="./index.html"><img src="../../assests/logo-primary.png" alt="logo" class="navbar-logo"></a>
        </div>
        <a class="open-menu font-size-6"><i class="fas fa-bars"></i></a>
        <a class="close-menu font-size-6"><i class="fa fa-times"></i></a>
        <div class="navbar-items-container">
            <div class="navbar-search-container">
                <input type="text" class="navbar-search" placeholder="What you are looking for?" />
                <i class="fas fa-search navbar-search-btn"></i>
            </div>
            <ul class="navbar-links">
                    <li><a href="./navbar.html"><i class="fas fa-user"></i>Account</a></li>
                    <li><a href="./navbar.html"><i class="fas fa-heart"></i>Wishlist</a></li>
                    <li><a href="./navbar.html"><i class="fas fa-shopping-cart"></i>Cart</a></li>
                </ul>
            </div>
        </nav>

JavaScript


        const openMenu = document.querySelector(".open-menu")
        const closeMenu = document.querySelector(".close-menu")
        const mainMenu = document.querySelector(".navbar-items-container")

        openMenu.addEventListener("click",()=>{
            openMenu.style.display = "none"
            closeMenu.style.display = "block"
            mainMenu.style.display = "flex"
            mainMenu.style.top = "74px"
        })
        
        closeMenu.addEventListener("click",()=>{
            closeMenu.style.display = "none"
            openMenu.style.display = "block"
            mainMenu.style.top = '-100%';
        })
