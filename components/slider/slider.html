Slider

Slider allow users to make selections from a range of values.

Simple Slider


        <label>
            <input class="filter-slider my-3" type="range" min="0" max="100" value="50" />
        </label>

Slider with Value

Slider Value changes as you increase or decrease it

0
100

HTML


        <div class="range-slider-container">
            <div class="slider-label">0</div>
            <label>
                <input class="slider-range" type="range" min="0" max="100" value="50" />
            </label>
            <div class="slider-value slider-label">100</div>
        </div>

Javascript


        const slider =document.querySelector(".slider-range")
        const sliderValue = document.querySelector(".slider-value")

        sliderValue.innerText = slider.value

        slider.addEventListener("input",()=>{
            sliderValue.innerText = slider.value
        })
